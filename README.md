## Try & Demo

Run:

```bash
$ git clone https://github.com/BaseMax/Flutter-Food-Ordering-App
$ cd Flutter-Food-Ordering-App
$ flutter run
```

Check tools:

```bash
$ flutter doctor -v
$ flutter devices
$ flutter pub get
$ flutter packages get
```

Build:

```bash
$ flutter build apk
```
